## 现象  
按 WASD 后角色没有移动。  
  
## 排查场景  
UE5  
Character 蓝图  
Enhanced Input  
  
## 可能原因  
1. 没有添加 Mapping Context  
2. 角色没有被 PlayerController 控制  
3. 输入事件没有触发  
4. 绑定错了蓝图  
  
## 我这次的真实原因  
BeginPlay 里没有 Add Mapping Context。  
  
## 解决方法  
在 PlayerController 或 Character 初始化时添加 Mapping Context。  
  
## 下次排查顺序  
1. 看输入事件有没有触发  
2. 看角色是否被 Possess  
3. 看 Mapping Context 是否添加  
4. 看移动逻辑是否执行
