
| 中文术语 | 英文            | 简单解释                                            | 归属 / 备注            |
| ---- | ------------- | ----------------------------------------------- | ------------------ |
| 烘焙   | Bake / Baking | 把原本需要实时计算的信息预先计算并保存下来，例如光照、法线、AO。               | 图形学 / Blender / UE |
| UV展开 | UV Unwrapping | 把三维模型表面摊平到二维平面上，让纹理可以正确贴到模型表面。                  | Blender / 建模流程     |
| 着色   | Shading       | 决定物体表面如何响应光照并呈现最终视觉效果的过程。                       | 图形学 / 材质           |
| 蓝图   | Blueprint     | Unreal Engine 的可视化编程系统，通过节点连接来实现逻辑。             | UE 核心概念            |
| 渲染   | Rendering     | 把三维场景中的模型、材质、灯光等信息生成屏幕画面的过程。                    | 图形学 / UE           |
| 材质   | Material      | 定义物体表面如何表现的一套规则，例如颜色、粗糙度、金属感。                   | 图形学 / UE           |
| 纹理   | Texture       | 提供给材质或模型使用的图像数据，用来描述颜色、法线、粗糙度等表面信息。             | 图形学 / 材质           |
| 资产   | Asset         | 游戏开发中可被工程使用和管理的资源统称，如模型、材质、纹理、蓝图、动画、音效。         | UE / 项目资源          |
| 关卡   | Level         | 游戏中的一个场景或地图单位，包含场景物体、灯光、逻辑等内容。                  | UE / 场景组织          |
| 组件   | Component     | 挂载在对象上的功能模块，用来给对象增加能力，如碰撞、相机、网格体、移动功能。          | UE 核心概念            |
| 对象   | Actor         | UE 世界中的基本对象基类，很多放进场景里的东西本质上都是 Actor。            | UE 核心概念            |
| 可控角色 | Pawn          | 一类可以被 Controller 控制的 Actor，通常用来表示玩家或 AI 可操控的对象。 | UE 核心概念            |
| 人物角色 | Character     | 一类常用于人物的 Pawn，通常自带胶囊体、骨骼网格体和角色移动组件。             | UE 核心概念            |
